# Bilibili_mobile
## 已使用的技术栈

- react hooks
- redux toolkit
- react-router-dom
- axios
- antd-mobile
- vite
  - path 优化路径选取
- 移动端 rem，vw 适配
- css in js 
  - styled-components 
  - classnames
- react-lazyload :图片懒加载
- prop-types :props 类型校验
- react-transition-group :路由切换过渡
- fastmock : 自定义后端接口

## 已实现的功能

1. 头部路由切换
2. 首页及动画视频列表展示
3. 视频详情展示
   - 没有实现对应视频源播放，使用图片代替
   - 底部包括视频信息、推荐视频列表和评论
   - 推荐视频和评论为轮播切换
4. 个人主页展示
5. 视频搜索功能
